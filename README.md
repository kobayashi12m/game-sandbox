# コア・ガード.io - ゲームコンセプト

## 基本概念

プレイヤーは中心にある **コア** とそれを守る **ガード（外殻）** を操作する新感覚の io ゲームです。

## ゲーム要素

### コア

- プレイヤーの中心部分
- **触れられたら即座に敗北**
- 移動は可能だが、常にガードに覆われている必要がある

### ガード（外殻）

- コアを包囲する防御層
- **自在に伸縮・変形可能**
- サイズを大きくすることができる唯一の要素
- 他プレイヤーのコアに触れることで勝利

## 勝利条件

**勝利**: 自分のガードで相手のコアに触れる
**敗北**: 自分のコアが相手のガードに触れられる

## 戦略要素

### 攻撃戦略

- ガードを伸ばして相手のコアを狙う
- 相手の隙をついてコアに接触する
- ガードのサイズを活かした包囲攻撃

### 防御戦略

- コアを完全に覆い隠す
- ガードの形状を工夫して死角を作らない
- 相手の攻撃を予測した回避行動

### 成長システム

- 他プレイヤーを倒すことでガードサイズが拡大
- より大きなガードで複数の敵を同時に攻撃可能
- サイズが大きくなるほどコアの防御も困難に

## ゲームプレイの特徴

- **攻守一体**: ガードは攻撃と防御の両方を担う
- **リスク・リワード**: 攻撃のために伸ばすとコアが露出するリスク
- **戦術的多様性**: ガードの形状変化による無限の戦術パターン
- **瞬間的勝負**: 一瞬の隙が勝敗を分ける緊張感

## 技術仕様案

- リアルタイムマルチプレイヤー対応
- 滑らかなガード変形アニメーション
- 直感的なマウス/タッチ操作
- シンプルながら戦略性の高いゲームバランス
